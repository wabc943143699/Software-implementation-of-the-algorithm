--------------------------------------------------------------------------------------------------------------------
Cordic_solve_cos_sin.m 以Matlab语言描述了用Cordic算法求解三角函数的迭代过程，该函数支持任意角度求解三角函数，不支持向量输入，
每次输入一个角度值和迭代次数，得到cos值和sin值输出。
